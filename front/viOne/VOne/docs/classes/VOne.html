assignLevelData

(
  • data
  • config
)

Assigns properties for multilevel/multicontained data arrays by traversing the complete array. Allows to set properties for last child elements and calculate total values for parents based on a given child value.

Parameters:

  • data Array

    The dataset to assign properties. Must have be in the following format:
    [ { keyName: keyname, valuesPropertyName: [values] }, ... { keyName: keyname, valuesPropertyName: [values] } ];

  • config Object

    Settings for generating the distribution

    • parentPropertyName String

      The keyName property (parent elements name). Default is 'key'.

    • childPropertyName String

      The values property name to search for for each parent. Default is 'values'.

    • setTotalBy Function | number

      Function to be evaluated | value to assign as totalValue for each last child element. Each parent will contain in the 'totalValue' property the sum of all children assigned values.

    • storeTotalValueAs String

      Used in conjunction with config.setTotalBy, indicates the property name to be assigned for totals. Each parent will hold this property as the sum of this property name of every child. Defaults to 'totalValue'

    • extendRecordsOn String

      If you want to generate records based on a given property, set this property with the property which value is the amount of records to append. The last child record will be copied as many times this property indicates.

    • processLastChildAs Function

      Function to be executed on every last child record. The function will receive the record for allowing you to make any aditional calculations and/or set new properties on/for it.

circlePackLayout

(
  • data
  • config
)

Creates a circle packed layout based on calculating a tangent circle for two previus tangent circles.

Parameters:

  • data Array

    The dataset to distribute.

  • config Object

    Settings for generating the layout

    • childPropertyName String

      The values property name to search for for each parent. Default is 'values'.

    • nestedData Boolean

      Indicates if data is nested. If set to true, childPropertyName must be set (unless child property name is 'values').

    • radiusPropertyName String

      The name for the property that stores (or will store) the radius on each element. Default is 'radius';

    • setRadius Function | Number

      A function that will receive each record from the data array for setting its corresponding center position. Must return a number. If not set, it's assumed that a radius property already exists.

    • positionPropertyName String

      The name that will store the center position of each record. Default is 'center'.

    • centerX Number

      X position for the first element. Default is 0.

    • centerY Number

      Y position for the first element. Default is 0.

    • padding Number

      Distance between circles. Default is 0.

    • toleranceDistanceFactor Number

      Tolerance for meassuring distances between circles to prevent overlapping. Defaults to 0.001

Creates a series of points almost evenly distributed among a sphere surface.

Parameters:

  • pointsCount Number

    Number of points to draw

  • startPosition THREE.Vector3

    Center of the sphere

  • radius Number

    Radius for the sphere

  • irregular Boolean

    If set to true, irregular-like distribution will be used.

Returns:

Float32Array:

An array of positions where x = index * 3, y = index * 3 + 1 and z = index * 3 + 2 (An array to be used as position attribute in a buffer geometry).

select

(
  • selectOptions
)
VOne.Selection

Selector for meshes added to a THREEJS scene via the @SceneManager.

Parameters:

  • selectOptions String

    The mesh(es) name(s) to be searched for. If the mesh of interest has set the name property, a '#' sign must be prepended. To search by the name assigned to the mesh when it was added to the scene via the @SceneManager, the selectOption must be prepended by underscore. If a class has been assigned to the mesh of interest, the selectOptions must be prepended by dot.

    Example:
    var whiteDotMeshes = VOne.select('.whiteDot');

    will return an object containing all the meshes that matches the class 'whiteDot'. Also, the oject will contain methods for setting position and scale animation options.

Returns:

VOne.Selection:

VOne.Selection Object containing all the meshes that matched the select options.

spiral

(
  • data
  • config
)

Creates a spiral layout based on the Archimedes Spiral by assigning a position for each record in the provided data set.

Parameters:

  • data Array

    The dataset to distribute.

  • config Object

    Settings for generating the layout

    • centerX Number

      The X center position. Default is 0.

    • centerY Number

      The Y center position. Default is 0.

    • sizePropertyName String

      The property name for each record. Default is radius. It can be set using config.setSize.

    • setSize Function | | Number

      A function that can be used to set the radius for each element.
      ex:
      config: { ...,
      setSize: function(element, index) {
      return element.total * 6.66;
      },
      ... }

    • positionPropertyName String

      The position property name where the result of each record in the data set will be stored.

    • linearPadding Number

      Distance between elements along the spiral. Default is 0.

    • radialPadding Number

      Padding applied each spiral cicle (proportional).

    • z Number

      Z position for the spiral. Default is 0.

    • looseSpiral Bool

      If set to true, padding params will be ignored and a loose spiral (Archimedes spiral) will be created.

    • startRadius Number

      Initial radius (a margin radius at the center). Default is 0.
