createShader

(
  • options
)
Object

Creates vertex and fragment shaders according to submited options.

Parameters:

  • options Object
    • [color] String optional

      Geometry' color property name. Default is undefined.

    • [varyingColor] Object optional

      An object containing varying color options. This property and options.color property are mutually exclusive. If an options.color object has been provided, it will be used for rendering color.

      • startColor String
        The geometry startColor property name.
      • endColor String
        The geometry' endColor property name.
    • varyingAttributes Object

      Required attributes for varying colors, sizes or positions (position still not implemented);

      • age String
        The geometry' age property name.
      • duration String
        The geometry' change duration property name.
    • [alpha] String optional

      Geometry' alpha property name. Default is undefined and a value of 1.0 will be used.

    • [points] Boolean optional

      Must be set to true if you're going to use THREE.Points geometry.

    • [pointsFactor] Float optional

      Size factor to be applied to THREE.Points geometry points when using that type of geometry. Defaults to 600.

    • [varyingSize] Object optional

      An object containing startSize and endSize properties. Start size value must be less than end size value. (only available for Points geometries).

      • startSize String
        The name for the starting size attribute.
      • endSize String
        The name for the ending size attribue.
    • [useTexture] URL optional

      URL / path to texture file to use as texture.

    • [useTHREETexture] THREE.Texture optional

      A THREE.js texture to be used. Exclusive with options.use_texture.

Returns:

Object:

An object containing vertexShader, fragmentShader and uniforms properties ready to be used with a THREE.ShaderMaterial.
