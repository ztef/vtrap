VOne.GeometryModel Class

VOne geometry model. Allows to manage geometries' properties and interactions.

Constructor

VOne.GeometryModel

(
  • name
  • 3DObject
  • [dataModel]
  • [attributes]
)

Parameters:

  • name String

    A name to identify the model.

  • 3DObject Object

    THREE.Points or [THREE.Mesh].

  • [dataModel] Array optional

    An array of the data-model objects with properties for rendering them.

  • [attributes] Object optional

    Buffer attributes for the graphical representation.

    • [size] Float32Array optional
    • [color] Float32Array optional
    • [position] Float32Array optional

Allows to define actions to be executed when a point cloud element is intersected.

Parameters:

  • eventName String

    The name of the event to react to (mouse over, mouse out, mouse click).

  • action Function

    The action to be executed when the corresponding event is triggered.
