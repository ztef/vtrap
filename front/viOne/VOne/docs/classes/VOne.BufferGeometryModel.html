setInteraction

(
  • eventName
  • action
)

Allows to define actions to be executed when a point cloud element is intersected.

Parameters:

  • eventName String

    The name of the event to react to (mouse over, mouse out, mouse click).

  • action Function

    The action(s) to be executed when the corresponding event is triggered. The function will receive two params. First one is the datamodel object corresponding to the intersected scene object, the second will be the datamodel object index.

    Example:

     geometryModel.setInteraction('onMouseOver', function(element, index){
       // Do watever you want with the element and / or the model based on the index supplied.
     });
