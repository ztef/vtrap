VOne.GeometryModel

Generates the geometries from the provided model and atach them to the scene.

Parameters:

  • config Object

    Config object

    • interactive Boolean

      Indicates if the generated geometries will be interactive.

    • hasRelations Boolean

      Indicates if elements will have relations with their corresponding parent.

    • elementZPos Function

      Function that will be processed to determine the z position for each element. Must return a Float number. By default returns a random number between 0 and 1000

    • elementColor Function

      A method used to determine the element's color. By default returns 'pink'.

    • elementAlpha Float

      Elements alpha. Must have a value between 0 and 1.

    • texturePath String

      Path to an image to be used as texture for elements.

    • [relationsLinesColor] Function optional

      A function that must return an array with 2 values for collor for the corresponding relation lines. The first element of the array is equivalent to the elements vertex, the second one the color for the parent element vertex.

    • [relationsLineWidth] Number optional

      Line width for relations. Default is 0.5

    • [relationsLinesAlpha] Number optional

      Alpha for relations lines. Default is 0.9

  • scene VOne.SceneManager

    The scene to attach the generated Points geometry.

  • model Array

    An array containing model data.

  • [name] String optional

    The name for the generated geometry.

Returns:

VOne.GeometryModel:

The VOne.GeometryModel corresponding to the provided model.
