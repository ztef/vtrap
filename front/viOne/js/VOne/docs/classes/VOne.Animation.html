meshBasicAnimation

(
  • mesh
  • animationType
  • destinationVector
  • duration
  • [callback]
)

Starts a basic animation for a given THREE.Mesh.

Parameters:

  • mesh THREE.Mesh

    A mesh to animate.

  • animationType Int

    Animation type: VOne.PositionAnimation || VOne.ScaleAnimation.

  • destinationVector THREE.Vector3

    The destination vector.

  • duration Float

    Duration time for the animation specified in miliseconds. Default is 500.

  • [callback] Function optional

    Callback function to be executed once the animation is finished.

pointsBasicAnimation

(
  • points
  • index
  • animationType
  • destinationVector
  • duration
)

Starts a basic animation for a given THREE.Points object.

Parameters:

  • points THREE.Points

    The Points object to animate.

  • index Int

    The index corresponding to the point to be animated.

  • animationType Int

    Animation type: VOne.PositionAnimation || VOne.ScaleAnimation || VOne.AlphaAnimation.

  • destinationVector THREE.Vector3 | | Float

    The destination position vector in case of position animation. In case of scale animation, a float target value must be provided.

  • duration Float

    Duration time for the animation specified in miliseconds. Default is 500.
