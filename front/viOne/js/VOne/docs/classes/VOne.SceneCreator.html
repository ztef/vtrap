createScene

(
  • config
)
VOne.SceneManager

Creates a scene with specific parameters.

Parameters:

  • config Object

    Config options for the scene that will be created

    • width Number

      Sets the scene width. Defaults to window.innerWidth.

    • height Number

      Sets the scene height. Defaults to window.innerHeight.

    • bgColor THREE.Color

      Sets the background color for the scene. Default is 0x51008b.

    • bgAlpha Float

      Value betweeen 0 and 1 that defines the scene background alpha. Default is 1.

    • glRenderer Object

      Renderer to use in webGL scene.

    • verticalDegFOV Number

      Specifies the vertical FOV in degrees. Default is 45.

    • camera THREE.Camera

      Sets the camera instance to be used. If not provided, a new camera will be created.

    • cameraNear Float

      Sets the camera near to be used in the scene..

    • cameraFar Float

      Sets the camera far to be used in the scene.

    • controls THREE.Controls

      Specifies a camera controller (THREE.TrackballControls, THREE.OrbitControls, etc).

    • controlsConfig Object

      A custon config for the supplied controls.

    • useControls Object

      If you need to reuse a camera controls instance, use this param to provide the controls instance.

    • parentType String

      Container type element ('div' or 'window').

    • containerId String

      Allows to set an id for the scene.VOneData.container.

    • raycastThreshold Number

      Sets the scene.VOneData.raycaster threshold for points geometry.

    • useTweenAnimations Boolean

      If this is set to true, tween animations will be available to be used in the scene. Default is false. (Only available for WebGLRenderer).

    • glRendererCustomProperties Object

      A set of properties to be set on the WebGLRenderer.

    • bodyBGColor Hex

      Sets the color for the whole canvas for being used as page background color.
