VOne.LeapGrabControls Class

Grab controls for a VOne.SceneManager. Allows to control a scene using Leap Motion controller. This can be used with any other controller designed for THREE.js. When a hand/object is detected, the default THREE.js controller will be disabled. Once no hand is present in the Leap Motion sensing area, the default THREE.js controller will be re-enabled.

Constructor

VOne.LeapGrabControls

(
  • scene
  • config
)

Parameters:

  • scene VOne.SceneManager

    The VOne.SceneManager containing the camera to control using the Leap Motion controller.

  • config Object

    Options for configuring controls:

    • objectToControl THREE.Object3D

      The object to control with the Leap Motion controller. Defaults to the camera from the provided scene.

    • panSpeed Number

      Acceleration for panning. Defult is 0.02.

    • rotationSpeed Number

      Acceleration for rotation. Default is 0.05.

    • zoomSpeed Number

      Acceleration for zooming. Work in progress... Do not use or do it at your own risk.
