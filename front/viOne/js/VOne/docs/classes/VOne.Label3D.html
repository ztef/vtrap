VOne.Label3D Class

Creates 2D planes with text for using them in a 3D scene. The plane will contain a canvas with dimensions according to the text and config parameters provided.

Constructor

VOne.Label3D

(
  • text
  • config
)
THREE.Object3D

Parameters:

  • text String

    The text to create. Ideally, you should provide just one line of text. [TODO: change this parameter to accept an array, each element would be a line with text].

  • config Object

    Options for customizing the text.

    • fontFace String

      Font face to be used. Defaults to Arial Narrow.

    • fontSize Int

      Size to be used in the font. Default is 18.

    • borderThickness Float

      Border thickness to be used in the font. Default is 0.

    • backgroundColor Object

      Background color to be used with the text in the form { r: red, g: green, b: blue, a: alpha }. Default is { r:150, g:0, b:150, a:0.0 }

    • textColor Object

      Color to be used for the text in the form { r: red, g: green, b: blue, a: alpha }. Default is { r:30, g:0, b:190, a:1.0 }.

    • textAlign String

      Text align (left, center, right);

    • spriteMaterial Boolean

      Indicates if the returned object will use a sprite material (always facing to the camera). If not set or set to false, the returned object will be a plane containing the text.

    • lineLength Number

      Amount of max chars per line. Default is 27.

    • maxLines Number

      Max number of lines to be used. Default is 3.

    • lineHeight Number

      Lines height. Default is fontSize + 5.

    • canvasFixedWidth Number

      Texture canvas prefered width.

    • canvasFixedHeight Number

      Texture canvas preferred height.

Returns:

THREE.Object3D:

A plane containing the text with the provided parameters.
