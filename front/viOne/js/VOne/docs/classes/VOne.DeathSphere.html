generate

Object

Generates VOne.GeometryModels representing death spheres.

Parameters:

  • data Array

    A nested array.

  • config Object

    An object containing configuration properties:

    • childrenPropertyName String

      The name of the children array property for each parent element. Default is 'values'.

    • parentPropertyName String

      The parents name property. Default is 'key'.

    • valueProperty String

      The property name that stores each child and parent value to be used for width an height of every slice. If parents are missing this property, the sum of each child value property will be calculated. Default is 'totalValue'.

    • marginDegrees Number

      Degrees to be left as margin in the sphere top and bottom. Default is 20.

    • radius Number

      The sphere radius. Default is 500.

    • segmentsResolution Number

      Number of segments to be used for each slice generated. Default is 32.

Returns:

Object:

An object containing two VOne.GeometryModel objects. mainGeometryModel: contains the first level slices, secondaryGeometryModel: The second level slices. This geometry models have no mesh array property set.
