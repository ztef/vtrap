VOne.Polygon Class

Creates a 2D polygon model.

Constructor

VOne.Polygon

(
  • verticesNumber
  • centerX
  • centerY
  • radius
  • poligonRotated
)

Parameters:

  • verticesNumber Int

    Number of vertices for polygon.

  • centerX Float

    Center x polygon position.

  • centerY Float

    Center y polygon position.

  • radius Float

    Radius of the circle where the polygon will be circumscripted.

  • poligonRotated Boolean

    If set to true, the first vertex will be at 90 degrees position. Else, first vertex will be positioned a 0 degrees.
